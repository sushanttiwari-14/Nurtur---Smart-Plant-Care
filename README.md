# 🌿 Nurtur – Plant Care App

Nurtur is an iOS app that helps users identify plants using the camera and manage their plant care easily.

---

## 🚀 Features

- 📷 Scan plants using camera  
- 🌱 Identify plant using API  
- ➕ Add and manage plants  
- 💧 Track watering schedule  
- ✏️ Edit plant details  
- 🌦️ View weather info  

---

## 🛠 Tech Stack

- SwiftUI  
- MVVM Architecture  
- Async/Await  
- URLSession (API calls)  
- Camera Integration  

---

## ⚙️ Setup

1. Clone the repo  
2. Open in Xcode  
3. Add API keys in `Secrets.xcconfig`  
4. Run the app  

---

## 🔐 Security

API keys are stored in a local config file and are not pushed to GitHub.
